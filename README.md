# PluginEditImagen

App que abre una ventana que contiene un botón que cuando se pulsa abre la camara para poder hacer una foto que localiza las partes que hay texto o podemos estirar o deformar la imagen seleccionas, luego le podemos dar un filtro a la imagen y por último la inserta en la pantalla principal.

## Plugin Usado

cordova-plugin-camera
cordova-plugin-document-scanner
